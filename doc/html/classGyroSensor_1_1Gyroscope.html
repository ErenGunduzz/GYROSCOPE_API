◆ readITG3200()

ReturnValues Gyroscope::readITG3200 ( )

Read gyroscope data from ITG3200 sensor.

This function reads the X, Y, and Z axis data from the ITG3200 gyroscope sensor using the I2C interface. It first sets the I2C address of the sensor, then writes the register address to initiate the read, and finally reads the 6 bytes of data representing the gyroscope's X, Y, and Z axis values.

Returns
Return one of the status of ReturnValues enum.

◆ readITG3205()

ReturnValues Gyroscope::readITG3205 ( )

Read gyroscope data from ITG3205 sensor.

This function reads the X, Y, and Z axis data from the ITG3205 gyroscope sensor using the I2C interface. It first sets the I2C address of the sensor, then writes the register address to initiate the read, and finally reads the 6 bytes of data representing the gyroscope's X, Y, and Z axis values. The register addresses for the ITG3205 are the same as for the ITG3200.

Returns
Return one of the status of ReturnValues enum.
